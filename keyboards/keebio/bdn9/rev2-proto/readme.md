# BDN9 Rev. 2 Prototype Version

The BDN9 Rev. 2 prototype versions can be identified by looking near the USB-C port. One side of the port will say `Rev. 2`, while the other side has a marking of `-A` or `-B`. These two versions do not have an EEPROM chip on them and are not resettable via the `RESET` keycode.

Additionally, version A has no underglow RGB LEDs.
